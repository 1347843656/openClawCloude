## 📋 测试方法

### 步骤 1: 打开新会话
- 在 Web Chat 或其他渠道开启**新的对话会话**
- 确保是**不同的 session**（不是当前这个）

### 步骤 2: 问测试问题
从下方"测试问题列表"中选择一个问题

### 步骤 3: 评估回答
检查 AI 是否：
1. ✅ 识别出这是 Not A Bug
2. ✅ 报出正确的 Jira 单号（如 GD-72099）
3. ✅ 使用标准回复话术
4. ✅ 解释清楚原因

## 🧪 测试问题列表

### 测试题 1: 学习数据统计（必测）⭐

**问题：**
```
学习数据显示为 0 是怎么回事？老师有布置任务，学员也完成了，但是后台显示为 0
```

**期望回答：**
- 识别为 Not A Bug
- 提到 **GD-72099**
- 说明：按任务**开始时间**统计，非完成时间
- 示例：任务 1.30 开始，学员 2 月完成，算在 1 月

**评分标准：**
- ⭐⭐⭐ 完整回答（包含 Jira 单号 + 原因 + 示例）
- ⭐⭐ 部分正确（有原因但没单号）
- ⭐ 回答错误或不相关

### 测试题 2: 快速点评看不到学员

**问题：**
```
快速点评中看不到某个学员的任务，是怎么回事？
```

**期望回答：**
- 识别为 Not A Bug
- 提到 **GD-65412**
- 说明：快速点评列表逻辑是老师可见**他所在班级内**发布的任务
- 原因：老师和学员未共同存在于同一班级
- 建议：将老师添加到学员所在班级

**评分标准：**
- ⭐⭐⭐ 完整回答
- ⭐⭐ 部分正确
- ⭐ 回答错误
