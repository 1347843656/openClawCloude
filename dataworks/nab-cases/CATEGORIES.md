## 👤 账号管理

**特点：** 账号添加、删除、绑定、权限问题

**典型案例：**
- GD-55971: 绑定号码非新用户
- GD-63996: 账号系统全局唯一
- GD-65570: 账号在全局存在但不在该机构
- GD-67870/68305/71009: 账号被删除（查操作日志）
- GD-70787: 手机号绑定在已禁用账号上

**标准回复：**
```
【Not A Bug - 账号系统】
账号在全局系统中已存在，但不一定在该机构内。
请在"学员管理"中搜索账号，确认是否在其他机构/班级。
```

## 📝 任务/打卡

**特点：** 任务布置、打卡活动、完成度统计问题

**典型案例：**
- GD-54730: 任务内容重复（修改书前部内容导致）
- GD-65271: 模板记错/名称弄混
- GD-65935/67261/67275: 完成度/进度计算逻辑
- GD-67469: 打卡统计逻辑
- GD-72075: 收银宝配置问题

**标准回复：**
```

之后的新模板不会有该问题。
```

## 🏫 班级/权限

**特点：** 班级管理、数据可见性、快速点评问题

**典型案例：**
- GD-63997: 多机构消息问题
- GD-65412: 快速点评看不到学员（不在同一班级）
- GD-67310: 学员状态过滤问题

**标准回复：**
```
【Not A Bug - 班级权限】(GD-65412)
快速点评列表逻辑是老师可见他所在班级内发布的任务。
该老师和学员未共同存在于同一班级。
建议：将老师添加到学员所在班级。
```
