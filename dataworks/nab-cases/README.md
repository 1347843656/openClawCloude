## 🎯 使用指南

### 按问题类型查找

1. **数据统计问题** → 查看 `DataWorks` 分类
2. **账号相关问题** → 查看 `账号管理` 分类
3. **登录问题** → 查看 `登录认证` 分类
4. **任务/打卡问题** → 查看 `任务/打卡` 分类
5. **权限/班级问题** → 查看 `班级/权限` 分类
6. **账单/费用问题** → 查看 `账单/计费` 分类

### 快速回复流程

```
1. 收到问题 → 识别问题类型
2. 查找对应分类 → 匹配 Jira 单号
3. 使用标准回复话术 → 快速响应
4. 不匹配 → 收集信息 → 人工排查
```
