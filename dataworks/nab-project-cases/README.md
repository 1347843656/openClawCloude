## 🎯 重点案例回复话术

### GD-54730 - 打卡任务内容重复
```
【Not A Bug - 任务生成逻辑】
7 月 31 日修改了书前部的内容，导致后面生成 task 被往前推了 3 天。
之后的新模板不会有该问题。
```

### GD-57760 - 未展示分数
```
【Not A Bug - 功能限制】
该课文不支持测评，任务所选课文在 content 账号下没有测评标识。
```

### GD-63997 - 家长通发送提醒
```
【Not A Bug - 多机构】
学员同时在两个机构，收到的消息是另外一个机构的。
学员在当前班级内，没有离开动作。
```

### GD-65412 - 快速点评看不到学员
```
【Not A Bug - 班级权限】
快速点评列表逻辑是老师可见他所在班级内发布的任务。
该老师和学员未共同存在于同一班级。
建议：将老师添加到学员所在班级。
```

### GD-70805 - 示例学员有效期
```
【Not A Bug - 示例账号限制】
管理后台无法修改示例学员有效期。
已帮老师修改到 2099-01-01。
```

### GD-69911 - 扫码登录失败
```
【Not A Bug - 登录问题】
扫码登录问题，请确认：
1. app 版本是否最新
2. 网络连接是否正常
3. 是否已绑定机构
```
