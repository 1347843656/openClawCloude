# DataWorks NAB 工具使用说明

## 📁 文件结构

```
dataworks/
├── README.md           # 项目说明
├── nab-checklist.md    # 快速检查清单（人工查阅）
├── nab-classifier.py   # 自动分类工具（脚本）
└── usage.md            # 使用说明（本文件）
```

---

## 🚀 快速开始

### 方法 1: 使用分类器脚本（推荐）

```bash
# 进入目录
cd /home/admin/.openclaw/workspace/dataworks

# 运行分类器
python3 nab-classifier.py "错误信息或问题描述"
```

**示例:**
```bash
# 权限问题
python3 nab-classifier.py "表不存在，无权限访问"

# 调度问题
python3 nab-classifier.py "任务没跑，依赖没满足"

# 数据问题
python3 nab-classifier.py "分区不存在，数据没到"
```

**输出:**
- 匹配的 NAB 场景
- 建议回复话术
- 进一步检查项
- JSON 格式结果（便于集成）

---

### 方法 2: 查阅检查清单

直接打开 `nab-checklist.md`，对照症状查找：

1. 看**现象**列找到匹配的描述
2. 查看**原因**和**检查方法**
3. 复制**标准回复**话术

## 📋 支持的 NAB 场景

| 类别 | 场景 | 关键词 |
|------|------|--------|
| **权限** | 表不存在/无权限 | 表不存在，Access Denied，无权限 |
| **调度** | 任务未触发 | 任务没跑，没触发，not trigger |
| **调度** | 依赖未满足 | 依赖没，上游失败，dependency |
| **数据** | 数据延迟 | 数据没到，表是空的，data empty |
| **数据** | 分区不存在 | 分区不存在，partition not exist |
| **参数** | 参数替换失败 | 参数失败，${}错误，变量未定义 |
| **参数** | 业务日期不对 | 业务日期不对，bizdate wrong |
| **语法** | SQL 错误 | SQL 错误，syntax error，语法错误 |
| **语法** | 函数不支持 | 函数不存在，function not exist |
| **资源** | 资源不足 | 资源不足，排队，CU 不足 |
| **网络** | 连接超时 | 连接超时，timeout，连接失败 |

## 💬 标准回复模板

```
【Not A Bug - {问题类型}】

问题原因：{简短说明}
解决方法：{1-2-3 步骤}
参考文档：{链接}

如仍有问题，请提供：
- 任务 ID/节点 ID
- 错误截图
- 发生时间
```

## 🔄 添加新场景

遇到新的 NAB 场景时：

### 步骤 1: 编辑 nab-classifier.py

在 `NAB_RULES` 列表中添加新规则：

```python
{
    "name": "场景名称",
    "patterns": [r"关键词 1", r"关键词 2"],
    "category": "类别",
    "response": "标准回复话术",
    "checks": ["检查项 1", "检查项 2"]
}
```

### 步骤 2: 更新 nab-checklist.md

在对应的类别表格中添加新行。

### 步骤 3: 测试

```bash
python3 nab-classifier.py "测试关键词"
```
