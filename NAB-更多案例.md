## 📝 任务/打卡类

### GD-65660 - AI 自动点评
**现象：** 老师反馈说没有点 AI 点评，但前两天 AI 自动点评了  
**原因：** AI 点评功能逻辑（可能是批量点评或自动触发）  
**回复：**
```
【Not A Bug - AI 点评逻辑】
AI 点评功能支持批量点评和自动触发。
请确认：
1. 是否开启了自动点评设置
2. 是否有其他老师进行了点评
3. 查看点评时间戳确认具体操作时间
```

### GD-67261 - 任务完成度只有 94%
**现象：** 学员本月的作业都完成了，但是任务完成度只有 94%  
**原因：** 完成度计算逻辑（可能包含其他维度的统计）  
**回复：**
```
【Not A Bug - 完成度计算】
任务完成度计算包含多个维度：
1. 已完成任务数
2. 任务质量评分
3. 按时完成率
请查看详细统计报表确认具体计算方式。
```

**原因：** 扫码登录问题（版本/网络/绑定机构/设备）

## 🏫 班级/权限类

### GD-63997 - 家长通发送提醒
**现象：** 已经退出了班级，家长通仍发送作业提醒  
**原因：** 学员同时在两个机构，收到的是另一机构消息  
**回复：**
```
【Not A Bug - 多机构消息】(GD-63997)
学员同时在两个机构，收到的消息是另外一个机构的。
学员在当前班级内，没有离开动作。
请在"学员管理"中确认学员所属机构和班级。
```

### GD-65412 - 快速点评看不到学员
**现象：** 快速点评中看不到学员的任务  
**原因：** 老师和学员未共同存在于同一班级  
**回复：**
```
【Not A Bug - 班级权限】(GD-65412)
快速点评列表逻辑是老师可见他所在班级内发布的任务。
该老师和学员未共同存在于同一班级。
建议：将老师添加到学员所在班级。
```
