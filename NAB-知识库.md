## 🎯 核心用途

当用户报告 bug 时，先检查是否是 **Not A Bug**（不是真 bug，而是用户理解偏差/功能限制/统计口径等问题）。

**匹配到 NAB → 直接给标准回复，无需深入排查**

## 🔥 Top 10 高频案例

### 1. GD-72099 - 学习数据显示为 0
**现象：** 老师有布置任务且学员完成，但学习数据显示为 0  
**原因：** `dcp_teacher_task_count` 表按任务**开始时间**统计，非完成时间  
**回复：**
```
【Not A Bug - 数据统计口径】
学习数据统计是按【任务开始时间】归属月份的，不是按完成时间。
比如任务 1.30 开始，学员 2 月完成，也算在 1 月。
```

### 2. GD-54730 - 打卡任务内容重复
**现象：** 230 打卡不重复任务内容重复了  
**原因：** 7/31 修改了书前部内容，导致 task 被往前推了 3 天  
**回复：**
```
【Not A Bug - 任务生成逻辑】
7 月 31 日修改了书前部的内容，导致后面生成 task 被往前推了 3 天。
之后的新模板不会有该问题。
```

### 3. GD-57760 - 未展示分数
**现象：** 学员任务完成后，老师查看打卡任务时未展示分数  
**原因：** 该课文不支持测评  
**回复：**
```
【Not A Bug - 功能限制】
该课文不支持测评，任务所选课文在 content 账号下没有测评标识。
```

### 4. GD-63997 - 家长通发送提醒
**现象：** 已经退出了班级，家长通仍发送作业提醒  
**原因：** 学员同时在两个机构，收到的是另一机构消息  
**回复：**
```
【Not A Bug - 多机构】
学员同时在两个机构，收到的消息是另外一个机构的。
学员在当前班级内，没有离开动作。
```

### 5. GD-65412 - 快速点评看不到学员
**现象：** 快速点评中看不到学员的任务  
**原因：** 老师和学员未共同存在于同一班级  
**回复：**
```
【Not A Bug - 班级权限】
快速点评列表逻辑是老师可见他所在班级内发布的任务。
该老师和学员未共同存在于同一班级。
建议：将老师添加到学员所在班级。
```

### 6. GD-69911 - 扫码登录失败
**现象：** 版本是最新的，就是无法扫码登录  
**原因：** 扫码登录问题（版本/网络/绑定机构）  
**回复：**
```
【Not A Bug - 扫码登录】
请确认：
1. app 版本是否最新
2. 网络连接是否正常
3. 账号是否已绑定机构
4. 设备是否已绑定
```

### 7. GD-70805 - 示例学员有效期
**现象：** 示例学员到期无法调整有效期  
**原因：** 管理后台无法修改示例学员有效期  
**回复：**
```
【Not A Bug - 示例账号限制】
管理后台无法修改示例学员有效期。
已帮老师修改到 2099-01-01。
```

### 8. GD-55971 - 学员注册登录
**现象：** 机构没有开启自主注册，学员却可以注册登录  
**原因：** 绑定号码非新用户，用户名和手机号都可登录  
**回复：**
```
【Not A Bug - 账号绑定】
该账号只是绑定了号码并没有注册新用户。
用户名和绑定手机号都可以登录。
```

### 9. GD-63996 - 账号已存在
**现象：** 账号无法添加，提示账号已存在  
**原因：** 账号系统全局唯一  
**回复：**
```
【Not A Bug - 账号系统】
账号在全局系统中已存在，但不一定在该机构内。
请在"学员管理"中搜索账号，确认是否在其他机构/班级。
```

### 10. GD-65271 - 打卡模板缺失
**现象：** 打卡任务模板只能看到部分  
**原因：** 实际入库数量不足，老师可能记错  
**回复：**
```
【Not A Bug - 模板数量】
实际入库只有 X 个模板，老师可能记错或模板名称弄混了。
请在模板管理中确认实际数量。
```

   ↓
