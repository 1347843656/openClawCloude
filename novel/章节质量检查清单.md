# 章节质量检查清单

> 每章写作前后必须检查，确保逻辑自洽、设定一致
> _创建时间：2026-03-11_

---

## 写前检查（Opening Check）

### 1. 设定一致性
- [ ] 主角修为是否与上一章衔接？（当前：金丹中期）
- [ ] 主角权限是否正确？（当前：LV10）
- [ ] 能力使用是否有代价？（重影/时停→头痛流血）
- [ ] 地点是否正确？（当前：黑风城）
- [ ] 时间线是否连贯？（距离上一章过了多久）

### 2. 人物状态
- [ ] 林默：性格是否一致？（隐忍坚韧，不鲁莽）
- [ ] 王胖子：是否在场？作用是否合理？
- [ ] 苏清雪：位置/状态是否正确？（苍玄宗，被监视）
- [ ] 雷虎：立场是否模糊处理？（不直接揭示）
- [ ] 影子：意识残留，指导但不越界

### 3. 剧情推进
- [ ] 本章目标是否明确？
- [ ] 是否有至少一个冲突？
- [ ] 是否有至少一个推进事件？
- [ ] 章末是否有悬念？

### 4. 伏笔管理
- [ ] 本章要回收哪些旧伏笔？
- [ ] 本章要埋哪些新伏笔？
- [ ] 伏笔是否与主线相关？

---

## 写后检查（Closing Check）

### 1. 剧情冲突
- [ ] 是否有至少一个小冲突？（每 500 字一个）
- [ ] 冲突是否合理？（不突兀，有铺垫）
- [ ] 冲突是否解决或推进？

### 2. 悬念结尾
- [ ] 章末是否有钩子？
- [ ] 钩子是否吸引读者继续看？
- [ ] 钩子是否与下一章衔接？

### 3. 主线推进
- [ ] 本章是否推进了主线？
- [ ] 推进幅度是否合理？（不拖沓，不跳跃）
- [ ] 是否与前后章衔接？

### 4. 人物一致性
- [ ] 林默性格是否一致？（不突然变鲁莽/圣母）
- [ ] 配角是否有独立动机？（不是工具人）
- [ ] 对话是否符合人物身份？

### 5. 设定检查
- [ ] 修为体系是否一致？（炼体→筑基→金丹→元婴）
- [ ] 权限体系是否一致？（LV1-LV15+）
- [ ] 能力代价是否体现？（头痛/流血/透支）
- [ ] 世界观是否矛盾？（修仙 vs 科幻过渡自然）

### 6. 伏笔检查
- [ ] 旧伏笔是否正确回收？
- [ ] 新伏笔是否埋下？
- [ ] 伏笔是否记录到伏笔清单？

### 7. 字数与节奏
- [ ] 字数是否达标？（2500-3500 字）
- [ ] 节奏是否紧凑？（无注水）
- [ ] 对话是否过多/过少？（平衡）
